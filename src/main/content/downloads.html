Get Open Liberty

Get Open Liberty using these commands for Maven, Gradle, or Docker.

  • Maven
  • Gradle
  • Docker
<dependency>
<groupId>io.openliberty</groupId>
<artifactId>openliberty-runtime</artifactId>
<version>[19.0.0.9,)</version>

</dependency>
